A POST request can be generated by sending some soft of information to the server. For example, usually a POST request is involved when submitting a form or logging in to a website.

Both request and response headers can be found under the "Network" tab of developer tools. Click on a request and the headers are in the "Headers" tab. Each request has its own headers since they are separate requests.
